# IoT DDoS Detection using Machine Learning

##  Overview
This project detects IoT-based cyber attacks using machine learning techniques.
---
##  Features
- Data preprocessing and cleaning
- Feature selection
- Multiple ML models
- Clustering techniques
- Evaluation (ROC, Confusion Matrix)
---
##  Dataset
Due to large size, dataset is available here:
https://drive.google.com/drive/folders/1T7Mh_OpOG7tj4mrAEX5fKbVaQBBwAdom?usp=drive_link

##  How to Run
pip install -r requirements.txt
python main.py
